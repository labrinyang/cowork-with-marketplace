# cowork-with-marketplace

Plugin marketplace for the **cowork-with** family of Claude Code plugins.

## Add this marketplace

Run inside Claude Code:

```
/plugin marketplace add labrinyang/cowork-with-marketplace
```

## Available plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [cowork-with](https://github.com/labrinyang/cowork-with) | Jira agile workflow — issue tracking, sprints, epics, automated task lifecycle | 1.0.0 |

## Install a plugin

After adding the marketplace, run inside Claude Code:

```
/plugin install cowork-with@cowork-with-marketplace
```

## Roadmap

Future plugins in this marketplace:
- `cowork-with-confluence` — Confluence page read/write
- `cowork-with-github` — GitHub issue/PR workflow
- `cowork-with-linear` — Linear issue tracking (see [linear-cowork](https://github.com/labrinyang/linear-cowork))
